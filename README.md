# Machine Learning Coursework Projects

Репозиторий объединяет три домашних задания из курса по машинному обучению. Все решения оформлены в Jupyter Notebook и сопровождаются сохранёнными визуализациями в папке `plots/`.

## Структура

| Путь | Содержание |
| --- | --- |
| `Python_для_анализа_данных__Итоговыи__проект_Ноутбук-шаблон.ipynb` | Анализ резюме HeadHunter: полный EDA, очистка, подготовка признаков |
| `final_project_decision_tree_ensemble_clustering.ipynb` | Итоговый проект «От дерева решений к ансамблю и кластеризации» (дерево с нуля, бэггинг, PCA+k-means) |
| `practical_module_correlation_regression.ipynb` | Практический модуль по корреляции и регрессии (EDA, корреляции, OLS, бутстрэп, градиентный спуск) |
| `ExchangeRates.csv` | Курсы валют для нормализации зарплаты в задании HH |
| `practical_module_dataset.csv` | Датасет для практического модуля по регрессии |
| `plots/` | Подпапки с графиками для каждой из трёх работ (см. отдельный README) |

## Краткое описание работ

### 1. Анализ резюме HH
- Проверка типов/пропусков, очистка и преобразование колонок.
- Расчёт производных признаков (возраст, опыт, зарплата в рублях).
- Разведывательный анализ с интерактивными графиками Plotly (сохранены в `plots/task1/`).

### 2. Итоговый проект (дерево → ансамбль → кластеризация)
- Генерация синтетического датасета, реализация дерева решений с критерием Джини.
- Сравнение со `sklearn`, визуализация границ решений.
- Бэггинг на собственном дереве с графиком качества.
- Реализация k-means и PCA с нуля для Iris; сопоставление результатов со `sklearn`.
- Все ключевые изображения находятся в `plots/task2/`.

### 3. Практический модуль: корреляция и регрессия
- EDA и проверка распределений, корреляции Пирсона/Спирмена.
- Простая и множественная OLS, доверительные/предсказательные интервалы.
- Bootstrap для коэффициента Пирсона, градиентный спуск для одномерной регрессии.
- Графики (гистограммы, scatter, диаграммы остатков) лежат в `plots/task3/`.

## Воспроизведение
1. Установите зависимости: `pip install -r requirements.txt` (или установите перечисленные библиотеки вручную).
2. Запустите нужный ноутбук, предварительно убедившись, что необходимые CSV доступны по указанным путям.
3. Для перезагрузки графиков выполните все ячейки — изображения автоматически сохраняются в соответствующих подпапках `plots`.
