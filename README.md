# Проект: Анализ резюме из HeadHunter

## Описание проекта

Данный проект представляет собой анализ базы резюме, выгруженной с сайта поиска вакансий hh.ru. Основная задача - подготовить данные для построения модели, которая будет автоматически определять примерный уровень заработной платы, подходящей пользователю, исходя из информации, которую он указал о себе.

## Структура проекта

Проект состоит из четырех основных этапов:

1. **Базовый анализ структуры данных** - знакомство с данными, проверка типов, пропусков
2. **Преобразование данных** - извлечение и преобразование признаков в удобный для анализа формат
3. **Разведывательный анализ** - визуализация зависимостей и выявление закономерностей
4. **Очистка данных** - удаление дубликатов, обработка пропусков, удаление выбросов
