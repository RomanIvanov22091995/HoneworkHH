# Графики проекта

В этой папке сохраняются интерактивные графики Plotly в формате HTML.

## Как использовать

После выполнения ноутбука здесь будут созданы файлы:
- `plot_1.html` - Распределение возраста
- `plot_2.html` - Распределение опыта работы
- `plot_3.html` - Распределение заработной платы
- `plot_4.html` - Зависимость зарплаты от образования
- `plot_5.html` - Распределение зарплаты по городам
- `plot_6.html` - Зависимость зарплаты от готовности к переезду и командировкам
- `plot_7.html` - Тепловая карта зависимости зарплаты от возраста и образования
- `plot_8.html` - Диаграмма рассеяния опыта работы от возраста

Для просмотра графиков откройте HTML файлы в браузере.
