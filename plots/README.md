# Папка `plots/`

Здесь собраны результаты визуализации для трёх домашних заданий. Каждый подпроект складывает изображения в отдельную подпапку:

| Папка | Проект | Формат файлов |
| --- | --- | --- |
| `task1/` | Анализ резюме HeadHunter | HTML-графики Plotly (`plot_1.html` … `plot_8.html`) |
| `task2/` | Итоговый проект «От дерева решений к ансамблю и кластеризации» | PNG (`decision_boundaries.png`, `ensemble_accuracy.png`, `kmeans_pca.png`) |
| `task3/` | Практический модуль по корреляции и регрессии | PNG (`practical_histograms.png`, `height_weight_scatter.png`, `residuals_diagnostics.png`) |

Откройте нужный файл в браузере/просмотрщике, чтобы увидеть соответствующую визуализацию. Новые изображения автоматически пересоздаются при повторном выполнении ноутбуков.
